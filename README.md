# ❓ 常见问题

1. 整数和浮点数的绘制功能不完整，建议临时用 `ASCLL` 码来文本显示
2. 官方和本设计器角度定义不一样，后续版本修改
   - 官方：`0°` 指 `12` 点钟方向，**顺时针**绘制
   - 设计器：`0°` 指 `3` 点钟方向（X正半轴），**逆时针**绘制
3. 导入代码失败，后续版本修改
   - 导入原理是通过正则表达式匹配绘制函数的参数，如果参数类型不符合会匹配失败
   - 代码需格式化，缺少空格或者换行等也会匹配失败，格式参考导出的代码
